MARGIN, BORDER, & PADDING

The box below illustrates the elements, padding, border, and margin. If our element is text, it would occupy the blue box. The green padding represents the space between the text box and the border. The yellow border is the visible line that outlines the element, situated between the padding and the margin. The orange margin shows the empty space surrounding a text box, which prevents other elements from overlapping.

padding: 20px; Creates inner spacing between the text and the border.

border: 5px solid yellow; Adds a yellow outline around the box.

margin: 30px; Adds space around the element, separating it from others.
